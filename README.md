# demo-webdriverio

Mobile E2E testing project using **WebdriverIO** + **Appium** for Android and iOS.

## Quick Start

```bash
npm install

# Run tests on Android
npm run wdio:android

# Run tests on iOS
npm run wdio:ios
```
